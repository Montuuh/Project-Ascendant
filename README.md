# Project Ascendant

A fan-made roguelike deckbuilder set in the Pokémon world.

Your party of three Pokémon **is** your deck — each one contributes 4 moves to a shared hand. Every turn you decide which Pokémon leads, which moves to play, and when to swap. Swapping costs Action Points and shifts your hand, so every position change is a real decision.

Explore a procedurally generated region, recruit wild Pokémon, collect relics, and face a three-phase Gym Leader boss at the end of each run.

> Fan-made portfolio piece. Not affiliated with Nintendo or Game Freak.
